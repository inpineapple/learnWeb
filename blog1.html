HTML 입력창

2019.03.20

< input type="text"/>


입력창에 border를 설정할수 있다
input{ border: solid 3px teal; }

입력창에 파란 테두리가 생기는 것을 없애려면
input:focus { outline: none; }
*:focus { outline: none; }

CSS 요소들의 위치

2019.03.20

좌측메뉴고정
nav{ position:fixed;}

position 속성을 이용하면 요소를 겹치게 놓을 수 있습니다. 이때 요소들의 수직 위치를 z-index 속성으로 정합니다. 값은 정수이며, 숫자가 클 수록 위로 올라오고, 숫자가 작을 수록 아래로 내려갑니다.
nav{ position:fixed; z-index:2;}
